## Para rodar em seu computador

Para rodar o projeto local e preciso `node` e `npm`.
```
$ git clone https://github.com/EdisonCristovao/INE5421.git
```
Instalar os pacotes com `npm`.

```
$ cd INE5421
$ npm install
```
Executar usando o comando.
```
$ npm start
```
